# Contributing to Universal Music Bot

Terima kasih atas minat Anda untuk berkontribusi! 🎵

## 🚀 Quick Start for Contributors

1. Fork repository
2. Clone fork Anda: git clone https://github.com/YOUR_USERNAME/universal-music-bot.git
3. Install dependencies: pip install -r requirements.txt
4. Buat branch baru: git checkout -b feature/amazing-feature
5. Commit changes: git commit -m 'Add amazing feature'
6. Push ke branch: git push origin feature/amazing-feature
7. Buat Pull Request

## 📋 Development Guidelines

### Code Style
- Gunakan Python 3.8+
- Ikuti PEP 8 formatting
- Tambahkan docstrings untuk fungsi penting
- Gunakan type hints jika memungkinkan

### Testing
- Test bot dengan berbagai query musik
- Pastikan performance tidak menurun
- Test di Windows dan Linux jika memungkinkan

### Performance
- Bot harus support jutaan user concurrent
- Memory usage harus optimal
- Download speed harus maksimal

## 🎯 Areas for Contribution

- 🎵 **Music Sources**: Tambah platform musik baru
- ⚡ **Performance**: Optimasi speed dan memory  
- 🌐 **Localization**: Dukungan bahasa lain
- 🔧 **Features**: Fitur baru (queue, playlist, etc)
- 📖 **Documentation**: Perbaikan dokumentasi
- 🐛 **Bug Fixes**: Fix bugs dan edge cases

## 📝 Pull Request Process

1. Update README.md jika perlu
2. Update requirements.txt untuk dependencies baru
3. Test thoroughly sebelum submit
4. Jelaskan perubahan di PR description
5. Link ke issue yang terkait (jika ada)

## 🐛 Bug Reports

Saat melaporkan bug, sertakan:
- OS dan Python version
- Error message lengkap
- Steps to reproduce
- Expected vs actual behavior

## 💡 Feature Requests

Untuk request fitur baru:
- Jelaskan use case
- Berikan contoh implementasi
- Diskusikan impact terhadap performance

## 📞 Questions?

- Buat issue dengan label 'question'
- Join Discord (if available)
- Email maintainer

Happy coding! 🎶
